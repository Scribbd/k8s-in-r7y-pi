This is probably a comparison between apples and oranges. Both have

Jenkins is lower level and uses agents


|               | Ansible | Jenkins       |
| ------------- | ------- | ------------- |
| Language      | Python  | Java / Groovy |
| State         | Less    | Full          |
| Configuration | Local   | Database      |
| Agent         | No      | Yes           |
|               |         |               |
